Frequently Asked

How long are the generated names? expand_more
Tidyyy follows a semantic rule of 2-5 words per filename to maintain balance between context and file system brevity.
What happens if two files are similar? expand_more
We use an intelligent conflict resolution system. Instead of generic dates, we apply -2, -3 suffixes while maintaining the semantic root of the filename.
Will it slow down my computer? expand_more
No. Tidyyy is built in Rust with a minimal CPU/RAM footprint. It remains dormant until a new file triggers the watcher, utilizing idle cycles for inference.
